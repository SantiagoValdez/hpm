indexProyecto(request)

source code 

Funcion: Panel principal de administracion de proyectos

Parameters:
  • request - Objeto que se encarga de manejar las peticiones http.
Returns:
Si el usuario se encuentra logueado retorna un objeto HttpResponse del template proyectos.html renderizado con el contexto {'usuario': u, 'lista': lista}. Sino, retorna un objeto HttpResponseRedirect hacia '/login'.

eliminarProyecto(request, id)

Funcion: Se ocupa de eliminar un proyecto

Parameters:
  • request - Objeto que se encarga de manejar las peticiones http.
  • id - id del proyecto a ser eliminado.

nuevoProyecto(request)

source code 

Funcion: Se ocupa de crear un nuevo proyecto

Parameters:
  • request - Objeto que se encarga de manejar las peticiones http.
Returns:
Si el usuario se encuentra logueado y si un nuevo proyecto es creado exitosamente retorna un objeto HttpResponse del template proyectos.html renderizado con el contexto {'usuario' : u, 'lista' : lista, 'mensaje' : 'Se creo proyecto con exito'}. Sino, retorna un objeto HttpResponseRedirect hacia '/login'.

indexLineaBase(request, id_fase)

source code 

Funcion: Panel principal de administracion de las lineas base de la fase

Parameters:
  • request - Objeto que se encarga de manejar las peticiones http.
  • id_fase - Identificador de la fase del cual se visualizan sus lineas base.
Returns:
Si el usuario se encuentra logueado retorna un objeto HttpResponse del template lineasbase.html renderizado con el contexto {'usuario': u, 'fase' : fase, 'lineasb': lineasb}. Sino, retorna un objeto HttpResponseRedirect hacia '/login'.

eliminarLineaBase(request, id_fase, id_lineabase)

source code 

Funcion: Se ocupa de eliminar una linea base de una fase

Parameters:
  • request - Objeto que se encarga de manejar las peticiones http.
  • id_fase - Identificador de la fase de la cual se elimina la linea base. Utilizado para renderizar el indice de las lineas base.
  • id_lineabase - Identificador de la linea base a ser eliminada.
Returns:
Si el usuario se encuentra logueado y la linea base es eliminada exitosamente retorna un objeto HttpResponseRedirect hacia el indice de lineas base de la correspondiente fase. Sino, retorna un objeto HttpResponseRedirect hacia '/login'.

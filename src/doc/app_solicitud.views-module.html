indexSolicitud(request, id_fase)

source code 

Funcion: Panel principal de administracion de Solicitudes

Parameters:
  • request - Objeto que se encarga de manejar las peticiones http.
  • id_proyecto - El id del proyecto al que pertenece el comite.
Returns:
Si el usuario se encuentra logueado retorna un objeto HttpResponse del template solicitudes.html renderizado con el contexto {'usuario': u, 'lista': lista}. Sino, retorna un objeto HttpResponseRedirect hacia '/login'.

indexItem(request, id_fase)

source code 

Funcion: Panel principal de administracion de los items

Parameters:
  • request - Objeto que se encarga de manejar las peticiones http.
  • id_fase - Identificador de la fase del cual se visualizan sus items.
Returns:
Si el usuario se encuentra logueado retorna un objeto HttpResponse del template item.html renderizado con el contexto {'usuario' : u, 'fase' : fase, 'lista' : lista}. Sino, retorna un objeto HttpResponseRedirect hacia '/login'.

eliminarItem(request, id_fase, id_item)

source code 

Funcion: Se ocupa de eliminar un item de una fase

Parameters:
  • request - Objeto que se encarga de manejar las peticiones http.
  • id_fase - Identificador de la fase de la cual se elimina el item. Utilizado para renderizar el indice de los items.
  • id_lineabase - Identificador del item a ser eliminado.
Returns:
Si el usuario se encuentra logueado y el item es eliminado exitosamente retorna un objeto HttpResponseRedirect hacia el indice de items de la correspondiente fase. Sino, retorna un objeto HttpResponseRedirect hacia '/login'.

revertirItem(request, id_fase, id_item)

source code 

Funcion: Se encarga de revertir la version del item a una anterior

Parameters:
  • request - Objeto que se encarga de manejar las peticiones http.
  • id_fase - Identificador de la fase cuyo item sera revertido a una version anterior.
  • id_item - Identificador del item que sera revertido a una version anterior.
Returns:
Si el usuario se encuentra logueado y el item es revertido exitosamente retorna un objeto HttpResponse del template revivir-item.html renderizado con el contexto {'usuario' : u,'fase' : fase, 'item' : item, 'mensaje' : 'Se cambio la version del item con exito.' }. Sino, retorna un objeto HttpResponseRedirect hacia '/login'.

source code 

Funcion: Panel principal de administracion de los items

Parameters:
  • request - Objeto que se encarga de manejar las peticiones http.
  • id_fase - Identificador de la fase del cual se visualizan sus items.
Returns:
Si el usuario se encuentra logueado retorna un objeto HttpResponse del template item.html renderizado con el contexto {'usuario' : u, 'fase' : fase, 'lista' : lista}. Sino, retorna un objeto HttpResponseRedirect hacia '/login'.

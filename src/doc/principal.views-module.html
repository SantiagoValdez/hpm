Funcion: Genera la pagina principal

Parameters:
  • request - Objeto que se encarga de manejar las peticiones http.
Returns:
Si el usuario se encuentra logueado retorna un objeto HttpResponse del template home.html renderizado con el contexto {'usuario': u}. Sino, retorna un objeto HttpResponseRedirect hacia '/login'.
